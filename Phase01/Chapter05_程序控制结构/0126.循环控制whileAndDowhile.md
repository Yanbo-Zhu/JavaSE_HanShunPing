# while

## 基本语句

```java
    循环变量初始化；
    while(循环条件){

            循环体（语句）；

            循环变量迭代；

    }
```

## 注意事项和细节

循环条件是返回一个布尔值的表达式；

 while循环是先判断再执行语句。

# Do..while

## 基本语法

```java
    循环变量初始化；
    do{

            循环体（语句）；

            循环变量迭代；

    }while(循环条件)；
```

## 说明

- do while 是关键字；
- 也有循环四要素，只是位置不同；
- 先执行，再判断，就是说至少执行一次；
- 最后有一个分号。

## 注意事项和细节说明

循环条件是返回一个布尔值的表达式；
do...while循环是先执行，再判断，因此至少执行一次。
