# 进制介绍

对于整数，有四种表示方式：
二进制：0,1 ，满2 进1. <font color=red>以0b 或0B 开头</font>。
十进制：0-9 ，满10 进1。
八进制：0-7 ，满8 进1. <font color=red> 以数字0 开头表示</font>。
十六进制：0-9 及A(10)-F(15)，满16 进1.  <font color=red>以0x 或0X 开头表示</font>。此处的A-F 不区分大小写。

//n1 二进制
int n1 = 0b1010;

//n2 10 进制
int n2 = 1010;

//n3 8 进制

int n3 = 01010;

//n4 16 进制
int n4 = 0X10101;
